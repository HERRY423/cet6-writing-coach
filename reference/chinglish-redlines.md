# 中式英语黑名单（看到立刻替换）

阅卷老师对这些表达**已经审美疲劳到反感**——很多年的烂大街万能模板。

## 一、开头黑名单（最致命）

| ❌ 中式 | ✅ 替换思路 |
|--------|------------|
| Nowadays, with the rapid development of society/economy/science and technology, ... | 用具体现象描写代替。如："Walk into any classroom today, and you will see ..." |
| With the development of the Internet, ... | "As digital connectivity reshapes the rhythm of daily life, ..." |
| In recent years, more and more people ... | "A quiet but unmistakable shift is underway: a growing share of ..." |
| It is well known that ... / As we all know, ... | 直接陈述。如："Few would dispute that ..." 或干脆删掉 |
| Recently, the problem of X has attracted wide public attention. | "X has, of late, moved from the margins of public concern to its center." |
| There is no doubt that ... | "Beyond dispute, ..." 或直接陈述 |

## 二、过渡黑名单

| ❌ | ✅ |
|----|----|
| Firstly, ... Secondly, ... Thirdly, ... Last but not least, ... | To begin with / What is more / Equally telling / Above all |
| On the one hand, ... On the other hand, ... | Granted, X ... and yet, Y ... |
| Every coin has two sides. | 直接论述，不要谚语铺垫 |
| As a coin has two sides, ... | 同上，删 |
| As the saying goes, ... | 引用具体谚语前不要铺垫，直接引 |

## 三、论证黑名单

| ❌ | ✅ |
|----|----|
| Take Edison / Jobs / Ma Yun for example, ... | 用具体小例（一个统计数字、一个普通人故事） |
| It is reported that ... (后面跟个编的数据) | 要用就用合理数字；不会编就用 "Recent surveys suggest that ..." 弱化 |
| As an old saying goes, "Where there is a will, there is a way." | 这句话已死，换 "Persistence, as countless biographies attest, often outlasts talent." |
| Practice makes perfect. | "Mastery accrues only through repetition." |
| Knowledge is power. | "Knowledge, as Bacon long ago observed, confers a quiet authority." |

## 四、结尾黑名单（最致命第二）

| ❌ | ✅ |
|----|----|
| In a word, ... / In conclusion, we should ... | "Ultimately, ..." / "What this comes down to is ..." |
| It is high time that we did sth. | 句型本身没问题但用滥了，换 "The case for action is now overwhelming." |
| Let's join hands to ... | 删。不要 join hands |
| Only in this way can we create a better future. | 太空。换成具体可执行的呼吁 |
| Our country will have a bright future. | 删。空话 |
| All in all, every coin has two sides. | 双重死刑，立刻删 |

## 五、词汇/搭配黑名单

| ❌ | ✅ |
|----|----|
| more and more | a growing number of / an increasing share of |
| very + 形容词 | 换强度更高的单词（very important → crucial） |
| big problem | formidable challenge / pressing issue |
| do harm to | be detrimental to / undermine |
| have a good influence on | have a beneficial impact on / enrich |
| pay attention to | attend to / heed / take into account |
| make full use of | capitalize on / leverage |
| be good at | excel at / be adept at |
| be interested in | take a keen interest in / be drawn to |
| in modern society | today / in the current climate / 直接删 |
| as far as I am concerned | from my perspective / to my mind |
| broaden one's horizon | expand one's perspective / open one's eyes to |
| keep pace with the times | move with the times / stay abreast of |
| achieve one's dream | realize one's ambition / fulfill one's aspirations |

## 六、句型黑名单

| ❌ | ✅ |
|----|----|
| There be 句型连用三次以上 | 改用主动语态或抽象主语 |
| 全篇 SVO 简单句 | 至少加 2 个倒装、1 个独立主格 |
| Although ... but ... | 选一个：要么 Although，要么 but |
| Because ... so ... | 同上 |
| 用问句开头但接着不回答 | 反问要 ε 自答或在文末呼应 |

## 七、"安全但不出彩"的灰色地带（够用但拉不开分）

这些不算错，但只能上 8 分，想冲 11+ 要替换：

- people → individuals / one / 具体人群
- thing → matter / aspect / factor
- way → approach / avenue / strategy
- get → obtain / acquire / glean
- think → contend / maintain / hold

---

## 最后一条铁律

**写完通读一遍**，如果有任何一句话让你觉得"这话好像背过"，就**重写它**。
让阅卷老师看到一个**活生生在思考的人**，而不是一个**背了模板的考生** —— 这是 11 → 14 的灵魂差距。
