# 高频烂大街词替换库（含搭配 + 语域）

每个原词给：替换词、典型搭配、用例区分、避雷。

---

## 形容词类

### important
- **替换**：crucial, vital, essential, pivotal, paramount, indispensable, instrumental, integral
- **搭配**：play a pivotal role in / be of paramount importance to / be indispensable to / be instrumental in shaping
- **区分**：crucial/vital（口语书面通用）；paramount/pivotal（议论文调子）；instrumental（强调"起到作用"）
- **❌ 避雷**：very important / important and necessary（重复）

### good
- **替换**：beneficial, advantageous, rewarding, fruitful, constructive, exemplary, commendable, sound
- **搭配**：have a beneficial effect on / prove rewarding / a sound argument / a commendable initiative
- **区分**：good people → admirable/upright；good idea → sound/sensible；good result → favorable/promising
- **❌ 避雷**：very good (excellent/outstanding)

### bad
- **替换**：detrimental, adverse, deleterious, pernicious, dire, alarming, regrettable
- **搭配**：have an adverse impact on / deleterious to one's health / dire consequences
- **区分**：bad effect → adverse/detrimental；bad habit → pernicious；bad situation → dire
- **❌ 避雷**：very bad → disastrous / catastrophic

### big / large
- **替换**：substantial, considerable, formidable, immense, profound, far-reaching
- **搭配**：a substantial portion / a formidable challenge / a far-reaching impact
- **区分**：big problem → formidable challenge；big change → profound transformation

### many / a lot of
- **替换**：a multitude of, an array of, a wealth of, a host of, countless, myriad, a considerable number of
- **搭配**：a wealth of opportunities / a host of factors / myriad challenges
- **❌ 避雷**：many many / lots of lots of

### difficult
- **替换**：daunting, formidable, arduous, taxing, intractable
- **搭配**：a daunting task / an arduous journey / an intractable problem

### interesting
- **替换**：compelling, captivating, intriguing, thought-provoking, illuminating
- **❌ 避雷**：very interesting → riveting / mesmerizing（看强度）

### easy
- **替换**：straightforward, effortless, accessible, intuitive
- **搭配**：an intuitive interface / readily accessible

### popular
- **替换**：prevalent, widespread, ubiquitous, in vogue, mainstream
- **搭配**：a ubiquitous phenomenon / a widespread practice

### new
- **替换**：novel, emerging, unprecedented, nascent, fledgling
- **搭配**：a novel approach / an emerging trend / nascent industries

## 动词类

### think
- **替换**：contend, maintain, argue, hold, posit, assert, suggest
- **搭配**：I would contend that ... / Many maintain that ... / It can be argued that ...
- **区分**：think (主观) → believe/feel；argue/contend (论证) → 议论文优选；posit (学术正式)

### show
- **替换**：demonstrate, illustrate, reveal, indicate, underscore, highlight, exemplify
- **搭配**：This case illustrates ... / The data underscore ... / Recent events have laid bare ...

### help
- **替换**：facilitate, foster, cultivate, nurture, bolster, contribute to
- **搭配**：foster creativity / cultivate critical thinking / bolster confidence
- **区分**：help do → enable/empower to do；help sth grow → nurture/foster

### get
- **替换**：obtain, acquire, attain, secure, glean
- **搭配**：attain a goal / glean insights from / secure a position

### use
- **替换**：employ, utilize, harness, leverage, draw upon
- **搭配**：harness the power of / leverage technology to / draw upon a wealth of experience

### change
- **替换**：transform, reshape, redefine, overhaul, alter
- **搭配**：fundamentally reshape / radically transform / overhaul the system

### improve
- **替换**：enhance, refine, augment, elevate, hone
- **搭配**：enhance one's capacity / hone one's skills / elevate the standard

### cause
- **替换**：give rise to, trigger, precipitate, engender, breed
- **搭配**：give rise to controversy / precipitate a crisis / breed resentment

### increase / grow
- **替换**：surge, proliferate, escalate, mount, gather momentum
- **❌ 避雷**：increase a lot → soar / skyrocket / surge

### reduce
- **替换**：mitigate, alleviate, curb, curtail, diminish
- **搭配**：mitigate the risk / curb consumption / alleviate suffering

### solve
- **替换**：address, tackle, grapple with, resolve, untangle
- **搭配**：tackle the issue head-on / grapple with the complexity of

### realize / find out
- **替换**：come to appreciate, recognize, discern, perceive
- **搭配**：come to appreciate the value of / perceive a pattern

## 名词类

### people
- **替换**：individuals, citizens, the public, society, our generation, youngsters/adults/professionals
- **❌ 避雷**：many people → 用 a growing number of individuals / an increasing share of the population
- **关键**：尽量具体化（students/workers/parents），泛指时才用 individuals

### thing
- **替换**：matter, issue, aspect, factor, element, dimension
- **❌ 避雷**：a lot of things → various factors / multiple dimensions

### problem
- **替换**：issue, challenge, dilemma, predicament, conundrum
- **区分**：social problem → social issue；hard problem → conundrum/dilemma；personal trouble → predicament

### way
- **替换**：approach, avenue, means, strategy, route
- **搭配**：a viable approach to / an alternative avenue to / a strategic means of

### chance
- **替换**：opportunity, prospect, possibility, window
- **搭配**：a golden opportunity / a window of opportunity / dim prospects

### idea
- **替换**：notion, concept, perspective, viewpoint, premise
- **搭配**：challenge the prevailing notion / embrace the concept of

### result
- **替换**：outcome, consequence, ramification, upshot, repercussion
- **区分**：good result → favorable outcome；bad result → adverse consequence / unintended ramification

### example
- **替换**：instance, case, illustration, embodiment
- **搭配**：a textbook case of / a striking illustration of / the embodiment of

### society
- **替换**：community, the public sphere, the social fabric, our collective life
- **❌ 避雷**：with the development of society → as society evolves / amid sweeping social change

## 副词类

### very
- **替换**：remarkably, strikingly, profoundly, decidedly, exceptionally, manifestly
- **❌ 避雷**：very very → 直接换强度更高的词（very good → excellent）

### always
- **替换**：consistently, invariably, perpetually, without exception
- **区分**：要看语气，positive 用 consistently；批评用 invariably/perpetually

### often
- **替换**：frequently, habitually, recurrently, time and again
- **搭配**：habitually overlook / time and again, we see ...

### maybe / perhaps
- **替换**：arguably, conceivably, ostensibly, in all likelihood
- **区分**：议论文优选 arguably（"可以说"），不要满篇 maybe

### a little
- **替换**：marginally, somewhat, to a modest extent
- **❌ 避雷**：a little bit（口语）

## 连接词（避免通篇 firstly / secondly / finally）

详见 `connector-ladder.md`，这里只列高频替换：

| 烂大街 | 替换 |
|--------|------|
| firstly | to begin with / for one thing / the first consideration is |
| secondly | what is more / on top of that / equally telling is |
| finally | not least / above all / crucially |
| in my opinion | from my perspective / to my mind / personally speaking |
| in conclusion | in essence / to bring this argument full circle / when all is said and done |
| because | given that / in light of the fact that / owing to |
| however | that said / be that as it may / yet |
| also | what is more / equally important / moreover, ... |

---

**使用提示**：
- 一篇 150 词作文里**精准替换 5-8 个**就够，不要全篇炫词，会让阅卷老师觉得堆砌。
- 替换前问自己：**搭配对吗？语域对吗？** 不对就用基础词，比用错高级词强。
